Detailed Description

Holds the data relevant to the I2C interface configuration.

The initialise callback must be set and run before any other operation.

enum I2C_RESPONSE(* initialise) (struct i2c_interface *self)

Callback to initialise the I2C interface.

This must be run before any other operations can be applied to the struct. This function must set the initialised flag of self to true if successful. Failure to do this will cause an error to occur and the system to enter a tight loop.

Parameters
selfThe struct for the interface being initialised
